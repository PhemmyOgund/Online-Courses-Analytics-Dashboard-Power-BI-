# Online-Courses-Analytics-Dashboard-Power-BI-
This project was driven by a key business question:

“What makes an online course successful—visibility or actual learning?”

Many educational platforms focus heavily on course views as a success metric. However, high visibility does not always translate into meaningful engagement or learning outcomes. I wanted to explore the gap between what users click on and what they actually complete or engage with.

To approach this, I designed the dashboard to highlight two critical perspectives:

Most Viewed Courses → Measures popularity and marketing effectiveness
Most Learned Courses → Measures true engagement and value delivery

During the process, I focused on:

Defining clear metrics for “views” vs. “learning/engagement”
Structuring the data model to allow accurate comparisons across courses and categories
Designing intuitive visuals that make it easy for business users to identify trends and outliers

One key insight from this analysis is that highly viewed courses are not always the most engaging, which suggests opportunities to improve course content, structure, or user experience.

From a business perspective, this dashboard helps stakeholders:

Identify which courses attract attention vs. deliver value
Optimize content strategy based on engagement, not just traffic
Make more informed decisions about course development and marketing

This project reflects my approach to analytics:

Start with a business question, define meaningful metrics, and build solutions that drive actionable insights.
