# Online-Courses-Analytics-Dashboard-Power-BI-
This project presents an interactive Power BI dashboard designed to analyze trends in online education. It focuses on identifying the most viewed courses and the most engaged (learned) courses, helping educational businesses understand user behavior and optimize their offerings.
🎯 Objectives
Identify top-performing courses based on views and engagement
Understand learner preferences and trends
Provide actionable insights for content strategy and business decisions
📂 Data & Features

The dashboard analyzes key metrics such as:

Course views
Course enrollments / engagement
Course categories
User interaction trends
📊 Dashboard Highlights
📈 Top Viewed Courses: Tracks courses with the highest visibility
🎓 Most Learned Courses: Identifies courses with the highest engagement
🗂️ Category Analysis: Shows performance across different subject areas
🔍 Interactive Filters: Allows users to explore data by category, course, or other dimensions
💡 Key Insights
Some courses attract high views but lower engagement, indicating potential content gaps
High-engagement courses highlight areas where learners find the most value
Category-level trends help businesses prioritize content development
🛠️ Tools & Technologies
Power BI
Data modeling and transformation (Power Query)
Data visualization and dashboard design
🚀 Business Value

This dashboard enables educational platforms to:

Improve course offerings
Increase learner engagement
Make data-driven decisions on content and marketing strategies
This project demonstrates how data visualization can be used to transform raw data into actionable insights, helping educational businesses better understand learner behavior and improve course performance.
